This is the current entire frontend of the Task and Image Annotation application. 
It's built with Next.js, React, and Tailwind CSS, by delivering a dynamic and high performance user experience.

## The issues I faced (And solved)

### Issue: The Drag and Drop Despair
When implementing the Kanban board it feels native and responsive is tricky. Many libraries i tried for that for managing complex state during drag and drop for tasks made it break the UI.
I solved it using the documentation and great open source. That alloweds me the smooth UI and logic for the drag and drop.





## Environment & Setup Instructions

**Node Version**: v24.14.1
**Python Version**: Python 3.12.10 (used by backend)

### Detailed Steps to Run:
1. **Clone the repo**:
   ```bash
   cd frontend
   ```
2. **Install dependencies**:
   ```bash
   npm install
 
3. **Run the development server**:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:3000/`.
